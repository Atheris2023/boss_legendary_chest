#### Boss Legendary Chest

legendary\_chest.txt file in locale/xx to drop chests aside



Example for legendary\_chest.txt:

Group	Reaper

{

 	Mob	1093

 	#	vnum	qnt.	chance

 	1	71607	1	50 # bronze

 	2	71608	1	35 # silver

 	3	71609	1	15 # gold

}



The cumulated chances need to be 100, in this case 50 + 35 + 15 = 100
